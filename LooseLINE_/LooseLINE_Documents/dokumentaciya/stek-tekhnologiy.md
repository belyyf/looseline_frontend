| **Компонент**    | **Технология**                             | **Назначение**               | **Обоснование**                                                                                                                                                                                                                                                                                            |
|------------------|--------------------------------------------|------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Frontend**     | React + TypeScript                         | UI, адаптивность, SPA        | Богатая экосистема компонентов и библиотек (Material-UI, Ant Design, Shadcn/ui) ускоряет разработку. TypeScript обеспечивает типизацию и снижает количество ошибок в production.                                                                                                                           |
| **Backend**      | Python + FastAPI                           | REST API, WebSocket          | FastAPI обеспечивает высокую производительность (на уровне Go/Node.js), встроенную генерацию OpenAPI документации и автоматическую валидацию данных Pydantic.                                                                                                                                              |
| **БД**           | PostgreSQL + SQLAlchemy                    | Хранение данных              | PostgreSQL -- enterprise-grade solution с ACID-гарантиями, поддержкой JSON/JSONB для гибких структур данных и отличной производительностью. SQLAlchemy обеспечивает надежный ORM layer, миграции через Alembic и комплексные возможности для работы с реляционными данными.                                |
| **Real-time**    | WebSocket + [Socket.io](http://Socket.io)  | Live-обновления, кэширование | WebSocket обеспечивает двусторонний low-latency канал связи. [Socket.io](http://Socket.io) добавляет автоматическое переподключение, fallback механизмы и масштабирование через namespace/rooms, необходимые для надежной синхронизации состояния приложения.                                              |
| **AI**           | [io.net](http://io.net)                    | Прогнозы и аналитика         | Предоставляет унифицированный доступ к множеству открытых моделей (Llama, Mistral, GPT) через единый API. Экономит затраты на облачные сервисы и снижает vendor lock-in благодаря поддержке различных провайдеров.                                                                                         |
| **Auth**         | Better-auth                                | Безопасность, аутентификация | Modern open-source решение с поддержкой OAuth, MFA, session management и PBKAC2/bcrypt для хеширования паролей. Избегаем сложности самостоятельной реализации криптографических функций и уменьшаем surface area для security vulnerabilities.                                                             |
| **Платежи**      | Stripe                                     | Финансовые операции          | Industry standard для SaaS с полной поддержкой множества платежных методов, встроенным PCI-DSS compliance и никаких требований к долгосрочным договорам. Stripe Connect позволяет легко масштабировать модель монетизации.                                                                                 |
| **Тестирование** | Jest + React Testing Library + Cypress     | QA                           | Jest обеспечивает unit-testing с встроенным mocking. React Testing Library фокусируется на тестировании поведения (не implementation), что повышает reliability тестов. Cypress предоставляет стабильный E2E testing без flakiness. Вместе покрывают весь спектр тестирования.                             |
| **DevOps**       | Docker + GitHub Actions + Nginx            | Деплоймент, CI/CD            | Docker гарантирует консистентность между development/staging/production окружениями. GitHub Actions встроена в GitHub и позволяет настроить автоматизированные workflows без отдельных инструментов. Nginx -- проверенный высокопроизводительный reverse proxy с низким overhead.                          |
| **Мониторинг**   | Sentry + Prometheus + Grafana              | Логирование и метрики        | Sentry автоматически отлавливает ошибки в runtime с контекстом (user data, breadcrumbs, session replay). Prometheus собирает метрики приложения, Grafana визуализирует данные и позволяет создавать alerting rules. Вместе обеспечивают полную observability без необходимости платных enterprise-решений. |
