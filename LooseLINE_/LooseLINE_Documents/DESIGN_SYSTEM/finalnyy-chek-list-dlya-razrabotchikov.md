---

При создании компонентов убедитесь:

* [ ] Использованы переменные вместо хардкодированных цветов

* [ ] Все размеры текста из набора (11, 12, 14, 16, 18, 20, 24, 30)

* [ ] Все интервалы из набора (4, 6, 8, 12, 16, 20, 24, 32...)

* [ ] Все радиусы из набора (4, 6, 8, 12, 9999)

* [ ] Используются предопределённые тени

* [ ] Анимации используют стандартные длительности (150ms, 250ms, 350ms)

* [ ] Тексты чёткие и читаемые (контраст >= 4.5:1)

* [ ] Интерфейс адаптивен (mobile-first)

* [ ] Фокус видим для accessibility

* [ ] Используется правильная иерархия заголовков (h1 > h2 > h3)
