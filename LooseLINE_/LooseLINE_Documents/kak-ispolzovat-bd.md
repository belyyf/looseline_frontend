# Сначала запускаем нашу БД

1. Скачиваем Docker

2. Запускаем и каждый раз перед разработкой его запускаем, иначе будет выдавать ошибку при вводе следующей команды: **docker compose up -d**

3. **Вводим команду:** docker compose up -d
